# AdGuard Home

DNS-level ad blocking and parental control.

## Access

- URL: `https://adguard.<domain>`
- Admin port: 3000

## DNS Configuration

### Server IP: `<your-server-ip>`
### DNS Ports:
- TCP/UDP: 53

### Upstream DNS (recommended):
- `https://dns.cloudflare.com/dns-query`
- `https://dns.google/dns-query`

## Clients

Configure your router or individual devices to use AdGuard:
- Router: Change DHCP DNS to server IP
- Devices: Set custom DNS to server IP

## Features

- Ad blocking
- Parental control
- Safe search
- DNS-over-HTTPS/TLS
